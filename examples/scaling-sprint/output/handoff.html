Key Findings

Multi-Session p003

Session-keyed Map<session_id, SessionState> with lazy initialization from hook payloads. Dashboard shows unified feed with session color-coding via hue-from-hash, pill-bar switcher for filtering. Heartbeat timeout (5min) handles crashed sessions.

Multi-Sprint p004

Sprint scanner reads sprints/ directory with fs.watchFile hot-reload. /api/sprints endpoint serves sprint list. Cross-sprint references use sprint-slug:claim-id format. Backwards compatible with --claims flag.

Repo Cartography p001 p007

Compiler-generated wheat-manifest.json is a topic map (not file tree). A single Read call replaces 3-5 Glob/Grep calls. Generated in <10ms (~6.7ms measured), ~5.8KB. Wired into compiler pipeline, non-fatal on failure.

Security Stack p005 p009 p010

Cookie auth (HttpOnly, SameSite=Strict) with 302-redirect to clean URL. CORS restricted to request origin. WebSocket transport for CF tunnel compatibility. CSRF tokens with auto-rotation and configurable grace period. All zero-dependency.

State Persistence p008

Atomic JSON writes (tmp+rename) to .farmer-state.json. Restores trust levels, session rules, activity logs on restart. Graceful shutdown via SIGTERM/SIGINT. 30s periodic auto-save. ~45 LOC.
