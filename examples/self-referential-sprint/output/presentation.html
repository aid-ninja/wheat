Architecture

Key Design Decisions

Pre-Computed Frames
Frames extracted from git at build time, not at runtime
Eliminates repo dependency for viewers. Stateless render function means any frame can be shown independently. [r042, p016]
Hybrid Sub-Framing
Batch commits auto-split into sub-frames
Commits with >3 claims get sub-frames grouped by topic. Preserves git fidelity while solving pacing. [x015, e010]
Compiler as Diff Engine
7-pass pipeline, 1000 claims in 59ms
Same compilation.json feeds dashboard, brief, PDF, presentation, and replay. Linear scaling confirmed. [e001, w010, r038]
Zero Dependencies
Server: 133 lines, no npm. Compiler: 685 lines, no npm. All output: inline CSS/JS. [p001, r014]
17-Command Suite
3 tiers: Sprint Lifecycle (10), Quality Assurance (4), Operations (3). Three feedback loops. [r013, r016]

Pedagogy

What We Learned About Teaching

The sprint's most important finding: we built the instrument but not the lesson plan.

The Gap
The visualization shows what happened but not why
No annotations explain Wheat concepts at key moments. No learning objectives defined. [r049, r050]
Echo Chamber
13 claims, all from a single source
Pedagogy is the only topic with mono-source evidence. Needs external validation.

Proposed Solutions

5 learning objectives designed for the replay viewer [r064]

Open Questions

What We're Not Sure About

These are the areas where evidence is thin or confidence is low.

  • Audience assumptions are unvalidated. "Self-explanatory without external docs" is asserted but never tested with a real user. [d003, r051]
  • Scope expanded significantly. Sprint grew from visualization into compiler features, accessibility, presentations, and replay. Productive, but risky under time constraints. [r039]
  • Confirmation bias in external validation. All 15 witness claims are confirmatory. No disconfirming evidence was sought. [r055, x023-x027]
  • Selection bias in evidence tiers. Technical claims reach "tested" easily; UX claims stay at "stated." Evidence is structurally uneven. [x010]
  • Replay file size. 1.8MB uncompressed (corrected from 763KB estimate). Gzips to ~180KB, but delta encoding was rejected as premature. [x020-x022]
