Research today is fragmented. Findings live in chat threads, docs, and someone's memory. Decisions get made without traceability.

Wheat makes every claim defensible — from first question to final brief.

Get Started in 3 Steps
1
Clone
git clone https://github.com/aid-ninja/wheat.git
cd wheat
2
Init
Open the folder in Claude Code and run
/init
Creates your sprint with a research question and seed claims.
3
Go
Start researching. The compiler runs automatically.
/research "your question"
